Save and manage Contact Form 7 submissions in your WordPress database with an easy-to-use admin interface.

== Description ==

Save Submissions to DB for Contact Form 7 automatically stores all form submissions in your WordPress database and provides a clean, user-friendly interface to view, manage, and export the data.

= Features =

* Automatically saves all Contact Form 7 submissions
* View submissions in WordPress admin
* Filter submissions by form
* View detailed submission information
* Delete unwanted submissions
* Handles file uploads
* Clean and intuitive interface
* Secure data handling
* No configuration needed


== Installation ==

1. Upload the plugin files to `/wp-content/plugins/cf7-save-to-db`, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Contact Form 7 must be installed and activated
4. All form submissions will automatically be saved to the database
5. View submissions under Contact > Submissions

== Frequently Asked Questions ==

= Do I need Contact Form 7 installed? =

Yes, this plugin requires Contact Form 7 to be installed and activated.

= Where can I view the submissions? =

After activation, you'll find all submissions under Contact > Submissions in your WordPress admin menu.

= Can I export the submissions? =

The free version doesn't include export functionality. This feature is available in the premium version.

= Is it compatible with my theme? =

Yes, the plugin works with any WordPress theme as it uses WordPress core functionality.
