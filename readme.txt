== Description ==

Save Submissions to DB for Contact Form 7 is a plugin that saves Contact Form 7 submissions to the WordPress database.

== Installation ==

1. Upload the entire `cf7-save-to-db` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **CF7 Submissions** menu in your WordPress admin screen.
